# 🎓 SkillSphere – Online Learning Platform

SkillSphere is a modern and responsive online learning platform where users can explore courses, view course details, and manage their profiles with secure authentication.

---

## 🌐 Live Demo

🔗 https://skillsphere-e-course-platform.vercel.app

---

## 💻 GitHub Repository

🔗 https://github.com/arifislam121416/skillsphere-E-Course-Platform

---

## ✨ Features

- 🔐 Secure Authentication (Better Auth)
- 📚 Browse All Courses
- 🔍 Search Courses by Title
- 🔒 Protected Course Details Page
- 👤 User Profile Page
- ✏️ Update Profile Feature
- 🔥 Trending Courses Section
- 🏆 Top Instructors Section
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- ⚡ Loading Spinner for better UX
- 🔔 Toast Notifications
- ❌ Custom 404 Not Found Page

---

## 🛠️ Tech Stack
